# מיצוי זכויות 360

כלי סיוע לפקידי ביטוח לאומי למיצוי זכויות אזרחים — זיהוי זכויות רלוונטיות לפי תרחישי חיים.

## דמו חי
👉 [https://aviad1840.github.io/mitzui-360/](https://aviad1840.github.io/mitzui-360/)

## טכנולוגיה
- React 18 + TypeScript
- Vite
- Tailwind CSS

## אבטחה
- אפס שמירת מידע — לא localStorage, לא cookies, לא database
- רענון דף = איפוס מלא
- כל הנתונים ב-React state בלבד

## הרצה מקומית
```bash
npm install
npm run dev
```

## פריסה
Push ל-main → GitHub Actions בונה ומפרסם אוטומטית ל-GitHub Pages.
